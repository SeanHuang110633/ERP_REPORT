期中專案流程圖總覽

以流程清單方式呈現三層互動式報表的方法呼叫鏈路

1) 程式啟動與第一層清單

從程式入口到 BKPF 第一層清單輸出。

  1. START-OF-SELECTION
  2. CREATE OBJECT go_report
  3. go_report->start()
  4. lcl_report->get_level1()
  5. lcl_report->display_level1()

2) 第一層雙擊 -> 第二層明細

AT LINE-SELECTION 觸發互動處理。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. gv_level = sy-lsind = 1
  4. lcl_report->get_level2()
  5. lcl_report->display_level2()

3) 第二層明細輸出與借貸平衡

顯示 BSEG + SKAT 明細並計算借貸差額。

  1. LOOP mt_level2
  2. Fault Injection (特定條件修改金額)
  3. 累加借貸差額 (S 加 / H 減)
  4. WRITE 明細 (HKONT / KUNNR 可下鑽)
  5. 輸出借貸平衡結果

4) 第二層點 HKONT -> 第三層科目主檔

依游標欄位分流到 SKA1 科目主檔稽核。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. gv_level = sy-lsind = 2
  4. GET CURSOR FIELD (HKONT)
  5. lcl_report->get_level3_skat()
  6. lcl_report->display_level3_skat()

5) 第二層點 KUNNR -> 第三層客戶主檔

依游標欄位分流到 KNA1 客戶主檔稽核。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. gv_level = sy-lsind = 2
  4. GET CURSOR FIELD (KUNNR)
  5. lcl_report->get_level3_kna1()
  6. lcl_report->display_level3_kna1()

6) 無效欄位點擊提示

點到非 HKONT / KUNNR 欄位時顯示提示訊息。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. gv_level = sy-lsind = 2
  4. GET CURSOR FIELD (其他欄位)
  5. MESSAGE '請點選藍色的 客戶代碼 或 科目代碼 進行下鑽。'

7) 第三層底層提示

進入最底層後顯示狀態列訊息。

  1. display_level3_skat()
  2. MESSAGE '已到達報表最底層' TYPE 'S'
  3. display_level3_kna1()
  4. MESSAGE '已到達報表最底層' TYPE 'S'
