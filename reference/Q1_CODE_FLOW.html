Q1 流程圖總覽

以流程清單方式呈現 Dialog 與互動式報表的呼叫鏈路

1) 程式啟動與第一層清單

載入 BKPF 傳票抬頭並輸出第一層互動清單。

  1. START-OF-SELECTION
  2. go_report->start()
  3. lcl_report->get_level1()
  4. lcl_report->display_level1()

2) 第一層雙擊 -> 第二層明細

AT LINE-SELECTION 進入互動處理。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. sy-lsind = 1
  4. lcl_report->get_level2()
  5. lcl_report->display_level2()

3) 第二層點 HKONT -> 第三層科目主檔

依點擊欄位下鑽 SKA1。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. sy-lsind = 2
  4. get_level3_skat()
  5. display_level3_skat()

4) 第二層點 KUNNR -> 第三層客戶主檔

依點擊欄位下鑽 KNA1。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. sy-lsind = 2
  4. get_level3_kna1()
  5. display_level3_kna1()

5) 第二層雙擊審核行 -> Screen 0100

Report 進入 Dialog 審核畫面。

  1. AT LINE-SELECTION
  2. lcl_report->handle_interaction()
  3. sy-lsind = 2 + gv_line_action = REVIEW_DOC
  4. prepare_review_dialog()
  5. CALL SCREEN 0100

6) Screen 0100 功能碼流程

SAVE / VIEW / CLEAR / BACK 的處理路徑。

  1. PAI: user_command_0100
  2. 保存: save_review()
  3. 查看摘要: CALL SCREEN 0200
  4. 清除: clear_review_input()
  5. 返回: LEAVE TO SCREEN 0

7) Screen 0200 轉出摘要報表

Dialog 轉回 list-processing 顯示摘要。

  1. PBO: status_0200
  2. SUPPRESS DIALOG
  3. LEAVE TO LIST-PROCESSING
  4. display_review_summary()
