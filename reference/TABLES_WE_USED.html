GBI FI Audit Report

SAP 標準表與 ER Model

本頁整理期中 FI 三層互動報表使用到的 SAP 標準表、關聯鍵與實際欄位。 報表從會計傳票抬頭出發，展開明細分錄，再依科目或客戶進入主檔稽核。

5 SAP 標準表
3 互動報表層級
2 第三層分流
RV 銷售相關傳票類型

使用表定位

每張表只取本專案需要的欄位，沒有使用整張表所有欄位。

BKPF

會計傳票抬頭表

Level 1

建立傳票清單與第一層下鑽索引。

BSEG

會計傳票明細表

Level 2

顯示每張傳票的借貸分錄與金額。

SKAT

總帳科目文字表

Level 2 Join

補上科目名稱，讓明細不只顯示代碼。

SKA1

總帳科目主檔表

Level 3 A

從科目代碼下鑽，檢查科目主檔屬性。

KNA1

客戶主檔一般資料表

Level 3 B

從客戶代碼下鑽，檢查交易對象資料。

ER Model 與下鑽關係

藍色是傳票抬頭，綠色是明細，黃色是文字表，紅色是主檔。

第一層到第二層

用 `BUKRS + BELNR + GJAHR` 從 BKPF 對應到 BSEG。

第二層補科目名稱

BSEG 的 `HKONT` 對應 SKAT 的 `SAKNR`，並限制語言與科目表。

第三層分流

點 `HKONT` 查 SKA1；點 `KUNNR` 查 KNA1。

欄位字典

藍色欄位是主要鍵值，綠色欄位是 Join / Drill-down 關聯欄位。

BKPF

會計傳票抬頭表
BUKRS 公司代碼，例如 `US00`；查詢條件，也顯示在第一層。
BELNR 會計傳票號碼；第一層主要顯示欄位，雙擊後查第二層。
GJAHR 會計年度；搭配公司代碼與傳票號碼定位資料。
BLART 傳票類型；本專案固定篩選 `RV` 銷售相關傳票。
BLDAT 文件日期；第一層顯示。
BUDAT 過帳日期；第一層顯示。

BSEG

會計傳票明細表
BUKRS 公司代碼；與 BKPF 一起定位傳票。
BELNR 傳票號碼；與 BKPF 一起定位傳票。
GJAHR 會計年度；與 BKPF 一起定位傳票。
BUZEI 傳票明細行號；第二層顯示每筆分錄序號。
HKONT 總帳科目代碼；第二層顯示，可連到 SKAT / SKA1。
SHKZG 借貸標示；`S` 借方、`H` 貸方，用於借貸平衡計算。
DMBTR 本位幣金額；第二層顯示並參與借貸加總。
MENGE 數量；第二層輔助顯示。
MEINS 數量單位；第二層輔助顯示。
KUNNR 客戶代碼；第二層顯示，可下鑽到 KNA1。
MATNR 物料代碼；第二層輔助顯示交易物料。

SKAT

總帳科目文字表
SAKNR 總帳科目號碼；與 `BSEG-HKONT` JOIN。
TXT20 科目短文字 / 科目名稱；第二層顯示科目名稱。
SPRAS 語言代碼；篩選目前登入語言，避免多語系重複。
KTOPL 科目表；本專案篩選 `GL00`。

SKA1

總帳科目主檔表
KTOPL 科目表；第三層顯示。
SAKNR 總帳科目號碼；用第二層的 `HKONT` 查詢。
XBILK 是否為資產負債表科目；`X` 表示是。
GVTYP 損益表科目類型；第三層顯示科目分類。
KTOKS 科目群組；第三層顯示科目群組。

KNA1

客戶主檔一般資料表
KUNNR 客戶代碼；用第二層的 `KUNNR` 查詢。
NAME1 客戶名稱；第三層顯示。
ORT01 城市 / 地區；第三層顯示。
LAND1 國家代碼；第三層顯示。
STRAS 街道地址；第三層顯示。
TELF1 電話；第三層顯示。
